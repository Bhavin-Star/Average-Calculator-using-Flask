Average Calculator using Flask
Description

A simple Flask-based web application that calculates the average of numbers entered by the user through an HTML form.
This project was built to understand Flask fundamentals such as routing, form handling, and backend logic using Python.

Tech Stack

Python
Flask
HTML
CSS (basic)

Features

Accepts multiple numbers as user input
Calculates the average on the backend
Displays the result dynamically on the webpage

Project Structure

Average-Calculator-using-Flask/
app.py
requirements.txt
templates/index.html
static/style.css

How to Run

Clone the repository

Install dependencies using:
pip install -r requirements.txt

Run the application using:
python app.py

Open the browser and visit:
http://127.0.0.1:5000/

Learning Outcome

Understanding Flask routing and request handling
Connecting HTML frontend with Python backend
Basics of web application development using Flask
